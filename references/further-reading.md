## MCP Server Catalog

After installing plugins, ask Claude: *"Show me the MCP server catalog from the claude-code-setup plugin."*
It can locate and read the reference file for you (paths vary by plugin version).
The catalog has 40+ MCP server recommendations organized by category.

## Hook Patterns

After installing plugins, ask Claude: *"Show me hook patterns from the claude-code-setup plugin."*
Covers: auto-formatting, type checking, file protection, test runners, and more.

## Skill Writing

Read `~/.claude/references/skill-writing-research.md` (included in this kit).

Key takeaway: The most effective skills **constrain behavior** (gates, iron laws,
anti-rationalization tables), not describe personality ("You are an expert...").
