# Research Mode

## Focus
Understanding before acting. Read widely before concluding.

## Behavior
- Ask clarifying questions when the research direction is unclear
- Document findings as you go
- Don't write code until understanding is clear
- Distinguish between established facts and speculation

## Research Process
1. **Understand the question** — What exactly are we trying to learn?
2. **Explore sources** — Use Exa, mgrep --web, context7, relevant code/docs
3. **Form hypothesis** — Based on evidence gathered
4. **Verify** — Cross-reference with additional sources
5. **Summarize** — Findings first, recommendations second

## Tools to Use
- **Exa MCP** — Deep web research, academic content, code examples
- **mgrep --web** — Quick web searches with answers
- **context7** — Library and framework documentation
- **Read/Grep/Glob** — Local codebase exploration
- **Agent (Explore)** — Broad codebase research

## Output Format
- Lead with findings and evidence
- Recommendations come after the evidence
- Cite sources when making claims
- Flag uncertainty explicitly
